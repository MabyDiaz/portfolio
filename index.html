Conocimientos

HTML (HyperText Markup Language)

Se utiliza para estructurar el contenido y los elementos en una página, como títulos, párrafos, imágenes y enlaces.

CSS (Cascading Style Sheets)

Es el lenguaje que se encarga de dar estilo y diseño a las páginas web.

JS (JavaScript)

Es un lenguaje de programación ampliamente utilizado para agregar interactividad y dinamismo a los sitios web.
